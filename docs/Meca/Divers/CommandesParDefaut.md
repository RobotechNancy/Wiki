# Commandes par defaut

En cas de suppression d'une entrée, il est possible de la re-ajouter facilement

- Ouvrir Candle
- Dans la bar du haut, ouvrir `Service/settings`
- Naviguer tout en bas dans `User commands`
- Cliquer sur `Remove all` puis sur `Import`
  <img src="/images/Meca/SetupCNC/UserCommandsControls.webp" alt="Boutons du panneau User commands" left>
- Sélectionner "Basics" et conserver les cases cochées qui vous intéressent

_And vouala_, vous avez récupéré les commandes qui peuvent vous intéresser

## Commandes par défaut

- `Zero XY`
- `Zero Z`
- `Z-probe`
- `Safe Z`
- `Work origin`
- `Machine origin`
- `Coordinates`

Plus d'informations dans `Help/Documentation` dans la section `Scripting/Examples/`
