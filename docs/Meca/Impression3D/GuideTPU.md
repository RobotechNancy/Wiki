# Guide pour imprimer avec du TPU

## TPU pour AMS

Facile comme tous les autres filaments, vous le mettez dans l’AMS et le système se fait automatiquement.

## TPU 95A et +

Utilisez le support de bobine arrière de la BAMBOU LAB P1S et mettez le filament dans le tube jusqu’à ce qu’il y ait une résistance, puis chargez le filament depuis l’interface de l’imprimante.

Pour la CR-10 faite comme avec tous les autre filament.

-	Pour des TPU plus fléxible : utilisée la CR-10 ou pour la P1S passée par au-dessus
