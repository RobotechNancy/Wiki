# Liste des outils à installer

## STM32

Le développement sur STM32 nécessite deux logiciels principaux:
- [STM32CubeMX](/Info/STM32/STM32CubeMX#Installation) : L'interface graphique permettant de sélectionner la carte utilisée et configurer les différentes broches.
- Un IDE : Soit [STM32CubeIDE](/Info/STM32/STM32CubeIDE#Installation) (un IDE basé sur Eclipse), soit VSCode avec l'extension [STM32CubeIDE for Visual Studio Code](/Info/STM32/VSCode#Installation).
